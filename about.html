About CareerQuest

Welcome to CareerQuest! This website helps users find job listings from universities and colleges across Canada.

The goal of CareerQuest is to make it easier for job seekers to find opportunities in one place, instead of searching through multiple university websites.

What This Website Does

CareerQuest collects job listings from various university and college websites using a web scraper. It then shows these listings on our platform, making it easier to find and apply for jobs.

How Our Scraper Works

The scraper is built using Python and Selenium. It automatically visits university websites, navigates to the job listings page, and extracts relevant job details such as the job title, description, and application link. The scraper runs on a regular schedule, ensuring that our job listings are always up-to-date. This powerful tool makes sure that users get the latest job opportunities without any manual effort.

Key Features

  • Comprehensive Listings: Find job postings from multiple universities and colleges in one place.
  • User-Friendly Interface: Easy to navigate and responsive design.
  • Advanced Search: Use the search and filtering options to find the perfect job.
  • Regular Updates: Get the latest job opportunities with frequent updates.

Technologies and Tools

  • Front-End: HTML, CSS, JavaScript for the user interface.
  • Back-End: Node.js and Express.js for server-side operations.
  • Design: Bootstrap for responsive design.
  • Data Handling: Web scraping using Python and Selenium.
  • Version Control: Git & GitHub for version management.
